 # Live News Dashboard 📰

Stay informed with cutting-edge technology news delivered fresh to your fingertips! This sleek, responsive web application brings you the latest tech headlines in real-time, all powered by the industry-leading NewsAPI.

Whether you're a tech enthusiast, developer, or just curious about what's happening in the digital world, this dashboard makes staying up-to-date effortless and engaging.

## 🚀 Live Demo
Check it out now: https://sameera0406.github.io/Live-News-Dashboard/

## ✨ Highlights
- **📡 Real-Time Updates:** Constantly fetches the top 20 freshest technology headlines from trusted sources worldwide
- **📱 Fully Responsive:** Seamlessly adapts to any screen size—mobile, tablet, or desktop—with a clean, modern UI powered by Pico CSS
- **⚡ Lightning-Fast:** Modern asynchronous JavaScript ensures smooth, non-blocking data fetching without lag
- **🎯 Developer-Friendly:** Built with vanilla JavaScript (ES6+), making the code clean, maintainable, and educational
- **🔌 API Integration:** Direct integration with NewsAPI for reliable, curated news content

## 🛠️ Tech Stack
| Technology | Purpose |
|-----------|---------|
| **HTML5** | Semantic structure and markup |
| **CSS3 (Pico CSS)** | Modern, lightweight styling framework |
| **Vanilla JavaScript (ES6+)** | Dynamic interactivity with async/await patterns |
| **NewsAPI** | Real-time news data source |
| **Fetch API** | Efficient HTTP requests |
